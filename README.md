# CNN-Based Waste Image Classification for Recycling

MSc Software Engineering – Pattern Recognition | Phase 2  
Student: Jasmine Praween

## Dataset
Garbage Classification Dataset (6 classes: cardboard, glass, metal, paper, plastic, trash)  
Dataset link: https://www.kaggle.com/datasets/asdasdasasdas/garbage-classification

## How to Run
1. Open the notebook on Kaggle: [Kaggle Notebook Link]
2. Add the Garbage Classification dataset as input
3. Enable GPU T4 x2 accelerator
4. Run All cells

## Models Used
- Custom CNN (built from scratch)
- MobileNetV2 (Transfer Learning)

## Results
- Custom CNN Accuracy: 24.45%
- MobileNetV2 Accuracy: 78.53%

- `waste_classification_fixed.ipynb` – Full notebook with CNN and transfer learning
